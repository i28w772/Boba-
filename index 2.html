Google Places API key (optional)
Add a key to enable real star ratings and review counts for nearby shops. Without a key, shop data comes free from OpenStreetMap (no ratings available).
YouTube Data API key (optional)
Add a key to search YouTube directly in the Videos tab. Without a key, you can still paste any YouTube link or video ID to watch it, and recently watched videos are saved for you.
Anthropic API key (for Boba AI)
Required to use the AI sidebar. Get one at console.anthropic.com. This key is sent directly from your browser to Anthropic's API and stored only in this browser's local storage — never on any server of ours. Anyone with access to this device/browser could read it from local storage, so only use a key you're comfortable having client-side, ideally one with a spend limit set.
